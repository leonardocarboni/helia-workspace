# Ciao Microagent

This microagent provides a friendly Italian greeting capability.

## Description

When activated, this microagent responds with "ciao" - a friendly Italian greeting that can be used both as a hello and goodbye.

## Usage

This microagent has no triggers and is available as a knowledge resource for the agent.
